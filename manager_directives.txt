**CRITICAL: Python Environment**
- Always use a virtual environment (`python3 -m venv venv && source venv/bin/activate`) for dependencies.
- Never run `pip install` globally.

**PRIORITY 1: Project Scaffolding**
- Create a `README.md` with:
  - Project description
  - Setup instructions
  - Architecture overview
- Create a `Makefile` with:
  - `make setup` (installs dependencies)
  - `make test` (runs tests)
  - `make clean` (removes temp files)

**PRIORITY 2: Qdrant Vector Database Setup**
- Use official Qdrant Python client.
- Document connection logic in `README.md`.
- Add error handling for DB failures.

**PRIORITY 3: MCP Server Framework**
- Define API endpoints in `openapi.yaml`.
- Use FastAPI or Flask (document choice in `README.md`).
